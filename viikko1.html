Toinen päivä

Miten netti toimii? Mikä on internetin ja WWW:n ero?

Internettiä käyttävät yksiköt, välillä kaapelit ja postikeskus. Internetin protokollat ovat enemmän kuin pelkkä www. Esim. sposti käyttää email-protokollaa. Internet itsessään toimii TPC/IP protokollan avulla. Koska IP osoitteita on vaikea muistaa, on domaineja (helpommin muistettava osoite IP:lle, joka on tässä pohjalla). Postikeskuksilla on tiedossa IP-osoitteet, keskus hakee koneelta sen internet sivun mitä haetaan ja palauttaa sivun tietokoneeseen. DNS nimipalvelimet muuttamat IP-osoitteen ihmisen luettavaksi

Mitä tarvitaan nettisivuja varten? Miten selain toimii?

Ensinnäkin tarvitsemme paikan missä nettisivut ovat: palvelin/ serveri. Ostetaan myös jostain domain (Esim. GRRR:llä Zoner). WWW on http-protokollalla (Hypertext, Transfet protocol) toimiva eli verkkosivukokonaisuus on joka on linkittynyt. Hyperteksti ei ole staattista tekstiä, vaan tarkoittaa että siinä on linkkejä mukana.

Selain poistaa sähköpostiosoitteen alkuosan (http/ https). Selain on esimerkiksi koneen laskimen tapaan ohjelma. Lähettää siis protokollan mukaisen get pyynnön ip osoitteeseen palvelimelle, Anna tämä html file joka on kytketty osoitteeseen. DNS lookup (kääntää domainin ip osoitteeksi). 404-sivu: HTML-fileä ei löydy. Kun file saatu, luetaan vasemmalta oikealle ylhäältä alas. Sitten rendaa HTML (leikkaa ei näkyväksi tarkoitetu pois näkymästä) Css tyyli määrittelee ulkoasua.
